# Underbelly Peak Hour Forecaster 🕒

A predictive tool designed to estimate wait times at the **Underbelly** eatery. This project uses fundamental Machine Learning to help students avoid long queues by identifying peak hours based on the day and time.

## 📌 Problem Statement
Predicting campus food court crowds is a common daily challenge. This project addresses the "real-world" problem of time management between lectures by providing a data-driven forecast of expected wait times.

## 🚀 Key Features
* **Synthetic Data Generation:** Simulates realistic campus foot traffic patterns (lunch rushes, evening spikes, and weekend trends).
* **Supervised Learning:** Implements a **Decision Tree Regressor** to map categorical days and hours to numerical wait times.
* **Ease of Use:** A single-sheet script that generates data, trains the model, and provides a sample prediction.
